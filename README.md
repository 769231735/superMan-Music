# vue-music，高级vue项目

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

在本机运行：
	1-请确保本机安装 node 
	2-下载之后,打开命令行cd至对应目录。
	3-输入npm install，下载依赖。
	4-下载完成之后，输入 npm run dev
	5-稍等片刻，自动在浏览器显示。可以用谷歌的手机模拟器下查看
